# Privacy policy

Last updated: April 2026.

Ovesta (“we”, “us”) describes how this demo site handles information. This is not legal advice. Update this text with counsel before production use.

## What we collect

- **Account data** you give at registration and profile and checkout: name, email, and optional shipping fields in this demo.
- **Analytics** are not enabled in the default build. If you add analytics, describe them here and align with consent where required.

## Cookies and sessions

We use an **HTTP-only session cookie** after you sign in so the site can remember your account on return visits. It is not readable by page scripts in the browser.

## Third parties

- **Catalog** on this demo comes from **DummyJSON**. Their service may log requests as any public API would.
- **OpenAI** may process messages if you use the product assistant and the server has an API key configured.

## Contact

Use the Contact page or **support@ovesta.com** for privacy questions about this demo.

## Changes

We may update this page. The “last updated” date will change when we do.
