# Return policy

Last updated: April 2026.

Ovesta is a **demo storefront**. No real orders ship, and no real refunds apply. Use this text as a starting point for a live store.

## Standard terms (template)

- **Window** You may request a return within **30 days** of delivery for many new, unopened items. Some categories may differ in a real policy.
- **Condition** Items should be in original packaging when possible. Opened products may have restocking rules in production.
- **Process** Start a return from your account or contact support. This demo does not process real returns.

## Demo note

Payments in this application are **simulated**. Nothing is charged, and nothing is shipped.
